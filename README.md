# Py-TwitterAPI-Sandbox

## Required Python packages:
- [Tweepy](https://www.tweepy.org/)
- [pandas](https://pandas.pydata.org/)
- [NumPy](https://numpy.org/)
```
pip install tweepy
pip install pandas
pip install numpy
```
